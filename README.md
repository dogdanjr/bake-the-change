# Bake the Change

Strategic plan hub for a 50+ bakery fundraiser in Rexburg, ID (Sept 12, 2026),
raising money for reusable feminine hygiene kits for school-aged girls in
Africa, benefiting Life Together Alliance and Village Threads.

"Bake the Change" is the working title — one of ten name options the event
organizer will choose from.

Single static page, no build step. Opens to a hub view with all 8
workstreams (sponsorship, bakery recruitment, volunteers, marketing,
logistics, storytelling, community partnerships, reporting) as clickable
tiles — click into any one for its full timeline, task list, and success
metric.
